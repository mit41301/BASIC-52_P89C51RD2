## P89C51RD2

NXP (founded by Philips) P89V51RD2  
The NXP (founded by Philips) P89V51RD2 is a 40MHz, 5 Volt 8051-based Microcontroller with 32 I/O lines, 3 Timers/Counters, 9 Interrupts/4 priority levels, 64K+8K FLASH, 1K on-chip RAM, SPI, Dual Data Pointers, WDT, 5-channel PCA.
